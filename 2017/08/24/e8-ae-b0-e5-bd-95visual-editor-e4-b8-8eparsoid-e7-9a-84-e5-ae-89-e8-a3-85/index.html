记录一下自己安装Mediawiki插件Visual Editor以及Parsoid的过程。

安装Node.js

Parsoid需要node版本大于4.x，因为Debian下默认node版本太低，需要更高的版本，索性安装最新的8.x。

curl -sL https://deb.nodesource.com/setup_8.x | bash -
apt-get install -y nodejs

通过node -v来确认版本

安装Parsoid

首先添加Mediawiki的源

apt-key advanced –keyserver keys.gnupg.net –recv-keys 90E9F83F22250DD7
echo “deb https://releases.wikimedia.org/debian jessie-mediawiki main” | tee /etc/apt/sources.list.d/parsoid.list

然后安装Parsoid，由于上一步添加的源是https地址，所以需要安装apt-transport-https包来让apt支持https。

apt-get install apt-transport-https
apt-get update && apt-get install parsoid

配置Parsoid

Parsoid配置文件位于/etc/mediawiki/parsoid/config.yaml，需要进行编辑。 修改第35行：

uri: ‘http://wiki域名/api.php'

Parsoid的默认端口是8142。 随后重启Parsoid:

service parsoid restart

安装Visual Editor

进入Mediawiki的extensions目录,使用git同步。

git clone https://gerrit.wikimedia.org/r/p/mediawiki/extensions/VisualEditor.git
cd VisualEditor
git submodule update –init

修改LocalSettings.php,加入以下代码启用Visual Editor：

$wgMFAutodetectMobileView = true;

// Enable by default for everybody
$wgDefaultUserOptions[‘visualeditor-enable’] = 1;

// Optional: Set VisualEditor as the default for anonymous users
// otherwise they will have to switch to VE
// $wgDefaultUserOptions[‘visualeditor-editor’] = “visualeditor”;

// Don’t allow users to disable it
$wgHiddenPrefs[] = ‘visualeditor-enable’;

// OPTIONAL: Enable VisualEditor’s experimental code features

#$wgDefaultUserOptions[‘visualeditor-enable-experimental’] = 1;
$wgVirtualRestConfig[‘modules’][‘parsoid’] = array(
// URL to the Parsoid instance
// Use port 8142 if you use the Debian package
‘url’ => ‘http://localhost:8142',
// Parsoid “domain”, see below (optional)
‘domain’ => ‘wiki域名’,
// Parsoid “prefix”, see below (optional)
‘prefix’ => ‘localhost’
);

至此Parsoid于Visual Editor安装成功并启用。
